# O que posso e não posso usar no LocalStack

Serviços incluídos no free tier:

    *DynamoDB - Banco de dados NoSQL 
    *Kinesis - Streaming de dados em tempo real 
    *S3 - Armazenamento de objetos para data lakes 
    *Lambda - Processamento serverless de dados 

*Serviços de Data Engineering que NÃO estão no free tier (requerem LocalStack Pro):
    *Glue - ETL e catálogo de dados 
    *Athena - Análise de dados com queries SQL 
    *EMR - Processamento de big data com Hadoop/Spark 
    *DMS - Database Migration Service 
    *MSK - Managed Streaming for Apache Kafka 
    *RDS - Bancos de dados relacionais
